Backtracking
Dado um grupo de N jogadores, numeradores de 1 a N, escrever todas as possíveis maneiras distintas de dividi-los 
em dois grupos não-vazios de tamanhos não necessariamente iguais. 
Ex.: Entrada: N = 4, Saída: 1 / 2,3,4 ; 2 / 1,3,4 ; 3 / 1,2,4 ; 4 / 1,2,3 ; 1,2 / 3,4 ; 1,3 / 2,4 ; 1,4 / 2,3
